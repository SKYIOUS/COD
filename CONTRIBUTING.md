# Contributing to COD

Welcome, and thank you for your interest in contributing to COD!

There are several ways in which you can contribute, beyond writing code.

## Asking Questions

Have a question? Open a [discussion](https://github.com/SKYIOUS/COD/discussions).

## Reporting Issues

Found a bug? Please [open an issue](https://github.com/SKYIOUS/COD/issues/new?template=bug_report.md).

### Look For an Existing Issue

Before creating a new issue, please search [open issues](https://github.com/SKYIOUS/COD/issues) to see if it's already been reported.

### Writing Good Bug Reports

File a single issue per problem. Include:

- COD version
- Your operating system
- Steps to reproduce
- What you expected vs what happened
- A code snippet or repository to reproduce

### Creating Pull Requests

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Run `npm run compile` to verify the build
5. Submit a PR against the `main` branch

### Final Checklist

- Search existing issues first
- Test with `--disable-extensions` to rule out extension conflicts
- Keep changes focused — one fix per PR

## Thank You

Your contributions make this project better. Thank you for taking the time to contribute.
